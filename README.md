## Develop

### Commands usage

| Task              | Command                       |
|-------------------|-------------------------------|
| Start application | `uv run fastapi dev`          |
| Lint code         | `uv run ruff check app/`      |
| Format code       | `uv run black app/`           |
| Run tests         | `uv run pytest`               |
